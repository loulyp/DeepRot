# DeepRot
DeepRot: A Deep Learning Approach for Classifying Brainrot (Hyper-Stimulative) Content

Deep learning model to classify short-form videos as **Brainrot** (hyper-stimulative) or **Non-Brainrot** using a CNN+GRU architecture.

## Abstract
Brainrot refers to cognitive decline from excessive low‑quality digital content. This project uses EfficientNetB0 for frame feature extraction and a GRU for temporal patterns, achieving 98% test accuracy.

## Dataset
- Brainrot dataset (Kaggle)
- Extra videos from Kinetics (5% subset)

## Requirements
- Python 3.12
- TensorFlow, OpenCV, scikit‑learn, etc.

## How to run
1. Open `DeepRot.ipynb` in Jupyter or Kaggle.
2. Adjust paths to your dataset.
3. Run all cells.

## Results
- Accuracy: 98%
- Precision/Recall/F1: 0.98 for both classes
